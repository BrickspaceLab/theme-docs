# Social icons


## Overview

A customizable social media icons block that displays a collection of social media links as icons. The block supports various social platforms including TikTok, Instagram, Twitter/X, Facebook, Pinterest, Snapchat, Tumblr, Discord, LinkedIn, Twitch, YouTube, and Vimeo.


## Common use cases

- Configure social media URLs in theme settings before adding this block
- Use appropriate gap sizes based on icon size selection
- Consider the number of social icons when choosing between stacked vs inline layout
- Adjust icon size (16-48px) based on the design context and available space
- Use padding options to properly space the block within its container


## Block settings

| Setting | Description | Options |
|---------|-------------|---------|
| Enable horizontal padding | Adds horizontal padding around the icons | Checkbox (default: false) |
| Enable top padding | Adds top padding around the icons | Checkbox (default: false) |
| Enable bottom padding | Adds bottom padding around the icons | Checkbox (default: false) |
| Gap size | Spacing between social icons | None, Default (default), Xs, Sm, Md, Lg, Xl |
| Font size | Controls the size of the social icons | 16 - 48 px (default: 20) |
| Layout | Controls the icon layout direction | Stacked, Inline (default) |
| Visibility | Controls when the block is visible | All (default), Mobile only, Desktop only |
