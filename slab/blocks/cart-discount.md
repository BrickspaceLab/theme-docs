# Discount input


## Overview

A discount code input block that allows customers to apply discount codes to their cart. Displays an input field with a button to apply the entered discount code.


## Common use cases

- Allow customers to apply discount codes in cart drawers or cart pages
- Ensure all translation keys are properly defined in locale files
- Configure button color to match your theme's design system
- Set appropriate padding based on surrounding content


## Block settings

### Spacing

| Setting | Description | Options |
|---------|-------------|---------|
| Enable horizontal padding | Adds horizontal padding around the discount input | Checkbox (default: false) |
| Enable top padding | Adds top padding around the discount input | Checkbox (default: false) |
| Enable bottom padding | Adds bottom padding around the discount input | Checkbox (default: false) |


### Color

| Setting | Description | Options |
|---------|-------------|---------|
| Button color | Controls the button color scheme | • Primary<br>• Secondary (default)<br>• Tertiary<br>• Neutral<br>• Plain<br>• Outline<br>• Inverted outline<br>• Blur |

| Setting | Description | Options |
