# Search summary


## Overview

A search results summary block that displays the number of search results found and the search terms entered by the user. The block only renders when search results are present and provides configurable styling options for text appearance, spacing, and layout.


## Common use cases

- Place this block at the top of search results pages for clear user feedback
- Use consistent styling with your search form and results layout
- Consider using center alignment for standalone search summary displays
- Enable max-width for better readability in wide layouts
- Adjust font size based on the importance of search feedback in your design
- Use appropriate padding to ensure proper spacing within search result containers


## Block settings

### Spacing

| Setting | Description | Options |
|---------|-------------|---------|
| Enable horizontal padding | Adds horizontal padding around the summary | Checkbox (default: false) |
| Enable top padding | Adds top padding around the summary | Checkbox (default: false) |
| Enable bottom padding | Adds bottom padding around the summary | Checkbox (default: false) |
