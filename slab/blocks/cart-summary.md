# Order summary

## Overview

A customizable content block that displays the current number of items in the cart and the subtotal price, used to provide users with a quick cart status summary.

## Common use cases

- Place this block in highly visible areas like headers or sticky navigation components
- Consider visibility settings when using on both mobile and desktop layouts
- Ensure translations are set up for 'labels.there_are', 'labels.items_in_cart', and 'labels.subtotal'

## Block settings

| Setting | Description | Options |
|---------|-------------|---------|
| Enable horizontal padding | Adds horizontal padding around the summary | Checkbox (default: false) |
| Enable top padding | Adds top padding around the summary | Checkbox (default: false) |
| Enable bottom padding | Adds bottom padding around the summary | Checkbox (default: false) |
| Text color | Controls the text color | Default (default), Alternative, Primary background, Primary foreground, Secondary background, Secondary foreground, Tertiary background, Tertiary foreground, Neutral background, Neutral foreground, Shade |
| Font family | Controls the font family | Standard (default), Heading, Subheading, Accent. Visible when font size contains type-- |
| Font size | Controls the font size | Smaller, Small, Default (default), Big, Bigger, Heading 1, Heading 2, Heading 3, Heading 4, Heading 5, Heading 6 |
| Enable max width | Limits the content width for better readability | Checkbox (default: false) |
| Horizontal alignment | Controls horizontal text alignment | Left (default), Center, Justify, Right |
| Visibility | Controls when the block is visible | All (default), Mobile only, Desktop only |
