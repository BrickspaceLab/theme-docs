# Inline

## Overview

A flexible inline container component that allows for horizontal display of child blocks. Supports both app and theme blocks flowing inline with customizable spacing and alignment.

## Common use cases

- Display multiple blocks inline (horizontally) for text with icons, inline images, or any content that should flow together
- Content wraps naturally when it reaches the container width
- Control the maximum height of inline block elements

## Child blocks

This block includes the following nested blocks:

- Rich text
- Image
- Icon
- Video

## Block settings

| Setting | Description | Options |
|---------|-------------|---------|
| Top spacing | Padding space at the top of the container | 0 - 300 px (default: 0) |
| Bottom spacing | Padding space at the bottom of the container | 0 - 300 px (default: 0) |
| Block max height | Maximum height for inline block elements (images, videos, SVGs) | 0 - 300 px (default: 30) |
| Visibility | Controls when the block is visible | All (default), Mobile only, Desktop only |
| Scroll animation | Adds scroll-triggered animations | None (default), Fade, Slide up, Slide down, Slide left, Slide right, Zoom |
