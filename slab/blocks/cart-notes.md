# Order notes

## Overview

A Shopify theme block component that displays a textarea for adding order notes to the cart.

## Common use cases

- Allow customers to add special instructions or notes to their orders
- Ensure all translation keys are properly defined in locale files
- Use in cart drawers or cart pages to capture order notes

## Block settings

| Setting | Description | Options |
|---------|-------------|---------|
| Label | Label text for the order notes checkbox | Rich text input |
| Enable horizontal padding | Adds horizontal padding around the notes field | Checkbox (default: false) |
| Enable top padding | Adds top padding around the notes field | Checkbox (default: false) |
| Enable bottom padding | Adds bottom padding around the notes field | Checkbox (default: false) |
| Visibility | Controls when the block is visible | All (default), Mobile only, Desktop only |
