# Shared discount


## Overview

A dynamic content block that displays shared discount messages. This block provides customizable text styling, layout options, and responsive visibility controls.


## Common use cases

- Use max-width setting for better readability of error messages
- Consider text color contrast when customizing the appearance
- Test error messages in both mobile and desktop views when using visibility options
- Display discount codes shared by customers or promotional messages


## Block settings

### Spacing

| Setting | Description | Options |
|---------|-------------|---------|
| Enable horizontal padding | Adds horizontal padding around the content | Checkbox (default: false) |
| Enable top padding | Adds top padding around the content | Checkbox (default: false) |
| Enable bottom padding | Adds bottom padding around the content | Checkbox (default: false) |
