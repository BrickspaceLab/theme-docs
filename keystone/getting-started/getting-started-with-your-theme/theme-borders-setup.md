# Theme Borders Setup

#### Before Starting

* Borders impact brand perception:
  * Rounded = Friendly
  * Sharp = Strong/Professional

#### Border Radius

1. **Button Radius**:
   * 0px = Sharp corners
   * 50px = Fully rounded
2. **Input Radius**:
   * Recommended: 0-10px
3. **Element Radius**:
   * Recommended: 0-10px

#### Border Width

* Standard: 1px
* Minimalist: 0px
* Bold: 2px
* Affects all border elements
